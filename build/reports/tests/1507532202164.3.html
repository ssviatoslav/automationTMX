Гарячі клавіші доступу в редакторі HTML

Дані дії доступні через гарячі клавіші:

Ctrl-O Змінити шрифт
Ctrl-P Змінити розмір шрифту
Ctrl-H Змінити формат тексту (заголовок і т.д.)
Ctrl-1 to Ctrl-6 Змінити рівень заголовку
Ctrl-= Змінити мову тексту для користувача або переробити мову під мультифільтр (це виникає після опції мульти внизу).
Ctrl-B Зробити текст жирним
Ctrl-I Зробити текст курсивом
Ctrl-U Підкреслити текст
Ctrl-S Закреслити текст
Ctrl-, Підписати
Ctrl-. Надписати
Ctrl-0 Очистити Word HTML
Ctrl-Z Відмінити
Ctrl-Y Вперед
Ctrl-L Вирівняти вліво
Ctrl-E Вирівняти по центру
Ctrl-J Вимкнути строку повністю
Ctrl-/ Текст йде зліва на право
Ctrl-| Текст йде з права на ліво
Ctrl-; Зменшити абзац
Ctrl-' Збільшити абзац
Ctrl-G Змінити колір тексту
Ctrl-K Змінити колір тла
Ctrl-F Пошук та заміна
Ctrl-` Переключитися на вид коду HTML
Ctrl-M Переключитися на режим редактора на весь екран
Ctrl-Alt-O Вставити
Ctrl-Alt-U Вставити список
Ctrl-Alt-R Вставити горизонтальну лінію
Ctrl-Alt-A Зафіксувати
Ctrl-Alt-L Зробити посилання
Ctrl-Alt-D Прибрати посилання
Ctrl-Alt-N Виключити автопосилання
Ctrl-Alt-I Вставити малюнок
Ctrl-Alt-T Вставити колонку
Ctrl-Alt-S Вставити смайлик
Ctrl-Alt-C Вставити спеціальний символ

Наразі є деякі обмеження:

  • Ми не можемо зараз використовувати Ctrl- гарячою клавішею для переключення виду коду HTML на вид WYSIWYG.
  • Немає гарячих клавіш для операцій з таблицями (доступними тільки в повноекранному режимі). Вони не працюють з клавіатурою, оскільки залежать від використання мишки для вибору клітинки в таблиці.
